#My own config for nvim
1. This config derived from nvim starter from TJ. Tweaks were made to make match my own programming style and frequently used language.
2. Added JDTLS for better integration with Java Spring
3. Added color picker and color viewer for color preview when working with CSS.
4. LSP for Typescript and Javascript when working with web dev
5. Few auto installed Mason with various tools like clangd for C and C++.
